Project Idea

The project idea I have come up with is a speedrunning tool that can automatically split the timer after completion of a segment in a video game, which allows the game user playing the game to have full concentration on the game without having to manually press a button to split the timer every time the game user completes a segment.

This idea is not big or interesting but a useful tool for game users who enjoy or take speedrunning video games professionally in which they require full concentration on the game. Having another game user to split or stop the timer for the game can result in inaccurate timing corresponding to the run of the category being a few milliseconds off what the final results should have ended with, as the person’s reaction time to the timing of the run can be a split-second delay which can impact the final record thus the game user’s ability to beat the other game user’s record.

The app will automatically split the timer upon detecting a frame from a video capture software that corresponds to the image sampled, which signifies that the segment of the game is complete. This can vary on a lot of games as to how the segment or level is completed, but for this project I will be using Mario Kart 8 Deluxe for the Nintendo Switch for the app to automatically split in which it needs to split the time upon seeing the “finish” sign on screen as soon as I finish a race. Because the app is AI based it needs to learn to detect the same message or frame on screen in game that would signify that the segment is complete multiple times before understanding the pattern. The app will have its own timer and profiles for specific games and categories that corresponds to the number of segments required to complete a speedrun. The app will automatically update the categories for world record runs to compare next to the user’s run.

The app will be programmed in C++ as well as the AI functionality. The software used for programming will be Notepad++. For displaying the video output of the game I will be using OBS. The hardware used for this project will be my laptop of course, a Nintendo Switch and an Elgato HD60S capture card to display the game’s video output to OBS.
